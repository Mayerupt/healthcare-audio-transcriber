# Healthcare Audio Transcriber

A desktop application that transcribes healthcare-related audio files into text using artificial intelligence.

## Motivation
Healthcare professionals often need accurate and private transcription of audio recordings. This project explores an offline AI-based approach to speech-to-text transcription with a focus on privacy.

## Tech Stack
- Python
- Whisper (Speech-to-Text)
- PySide6 (Desktop UI)
